# itrice-backend

## **Setup Backend**

#### 1. Access backend

```
cd backend
```

#### 2. Setup virtual environment

```
python -m venv venv
venv/Scripts/activate
```

#### 3. Install dependencies

```
pip install -r requirements.txt
```

#### 4. Run

```
flask run   
```

## API Documentation

Go to http://127.0.0.1:5000/swagger/ for documentation
